  • Dominio

    • En HTML, el "dominio" no es un término técnico específico. Generalmente, cuando se habla de "dominio" en el contexto de HTML, se refiere a la dirección web o URL que identifica un sitio web en internet.

  • Subdominio

    • Un subdominio en HTML no existe como tal. El término "subdominio" se refiere a una parte de un dominio web que se utiliza para organizar y segmentar contenido.

  • URL

    • En el contexto de HTML, una URL es la dirección web de un recurso, como una página web, una imagen o un archivo de audio.

  • DNS

    • El Sistema de Nombres de Dominio (DNS) es el "directorio telefónico de Internet" que traduce nombres de dominio legibles (como google.com) a direcciones IP (como 74.125.19.147) que las computadoras utilizan para comunicarse.

  • Hosting

    • El término "hosting" se refiere al servicio de alojamiento web, que permite a los usuarios de Internet almacenar y publicar sus archivos y contenidos en un servidor para que sean accesibles a través de la web.

  • Ftp

    • FTP, o File Transfer Protocol, es un protocolo de red que permite la transferencia de archivos entre un cliente y un servidor.
